ProBoard позволяет использовать mdx, что позволяет очень просто реализовывать идеи HR и рендерить файл как страницу сайта

## Даже джун сможет разобраться в коде, если он будет красиво подсвечен!

```tsx
export function cn(...inputs: ClassValue[]) {
  return twMerge(clsx(inputs))
}

export function formatDate(input: string | number): string {
  const date = new Date(input)
  return date.toLocaleDateString("en-US", {
    month: "long",
    day: "numeric",
    year: "numeric",
  })
}

export function absoluteUrl(path: string) {
  return `${env.NEXT_PUBLIC_APP_URL}${path}`
}
```

> Да да, здесь работают даже колауты

ProBoard поддерживает все базовые md тэги, а так же позволяет HR'ам создавать свои блоки
